# QuickGPT 🚀

QuickGPT is a full-stack AI-powered chatbot application built using the MERN stack (MongoDB, Express.js, React.js, and Node.js). It enables users to engage in intelligent conversations, generate AI-powered images, manage chat history, and securely authenticate accounts.

## Features

*  AI-powered text conversations
*  AI image generation from text prompts
*  User authentication and authorization
*  Persistent chat history storage
*  Credit-based usage system
*  Responsive and modern UI
*  Cloud-based image storage with ImageKit

## Tech Stack

* Frontend: React.js, Tailwind CSS
* Backend: Node.js, Express.js
* Database: MongoDB
* Authentication: JWT
* AI Integration: Gemini API
* Media Storage: ImageKit
* Payments: Stripe

QuickGPT is designed to provide a seamless AI experience by combining conversational AI and image generation within a single, user-friendly platform.
